# Video Submission Outline - Hello World Assignment

**Target Length:** 2-3 minutes  
**Assignment:** CSE310 Hello World Program

---

## Video Script & Timing

### [0:00-0:15] INTRODUCTION (15 seconds)
- "Hi, I'm [Your Name], and this is my Hello World program for CSE310."
- "I'll demonstrate my development environment and run the program."

### [0:15-0:45] CODE WALKTHROUGH (30 seconds)
- Open VS Code showing your project folder
- Briefly show `hello_world.py` file (5 seconds on screen)
- Point out: "Simple Python program with personalized greeting"
- Show `README.md` briefly (5 seconds on screen)

### [0:45-1:15] DEMONSTRATION (30 seconds)
- Open terminal in VS Code
- Run: `python hello_world.py`
- Show the output on screen
- Say: "Program successfully prints personalized Hello World message"

### [1:15-1:45] GITHUB REPOSITORY (30 seconds)
- Switch to browser with your GitHub repo
- Show repository page with files
- Say: "Code is version controlled and hosted on GitHub"
- Show commit history briefly

### [1:45-2:00] LEARNING REFLECTION (15 seconds)
- "Through this project I set up my development environment with Python, VS Code, and Git"
- "I'm ready to build more complex projects this semester."

### [2:00-2:15] CLOSING (15 seconds)
- "Thanks for watching. Link to repo is in the video description."
- Show repo URL on screen

---

## Pre-Recording Checklist
- [ ] Test run `python hello_world.py` to ensure no errors
- [ ] Ensure GitHub repository is public
- [ ] Have browser tab open to your GitHub repo
- [ ] Close unnecessary applications/windows
- [ ] Test microphone audio levels

## After Recording
- [ ] Upload video to YouTube
- [ ] Update README.md with actual YouTube link
- [ ] Add repository URL to video description
- [ ] Submit assignment with video link

## Recording Tips
- Use screen recording software (Windows: Xbox Game Bar with Win+G, or OBS Studio)
- Speak clearly but naturally
- Keep mouse movements slow and deliberate
- Pause between sections if needed - you can edit later
- No fancy editing required - straight recording is fine
